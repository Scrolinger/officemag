Блочные элементы

Блочные элементы задают основную структуру документа и логические части страницы. Они помогают разбить контент на смысловые фрагменты.

Основной каркас страницы

Главный раздел документа

Элемент <main> содержит главное содержимое страницы. Текущий текст как раз находится внутри main.

Вспомогательные блоки
Элемент div — нейтральный контейнер для группировки контента, без дополнительного семантического смысла.
Наиболее детальный уровень заголовка

Заголовки h1–h6 образуют иерархию документа: от главного заголовка до самых локальных подпунктов.

Строчные (inline) элементы

Строчные элементы используются внутри обычного текста и позволяют выделять, подчёркивать важное и добавлять семантику без разрыва строки.

Вот нейтральный span, ссылка на раздел со списками, просто жирный, стилистический курсив, подчёркнутый фрагмент, устаревшая информация, важное выделение и мелкий пояснительный текст.

Формулы: H2O и E = mc2 показывают использование <sub> и <sup>.

Списки

Неупорядоченные и упорядоченные списки

Маркированный список (ul) удобен для перечислений без жёсткого порядка:

  • Блочные элементы
  • Строчные элементы
  • Таблицы и формы

Нумерованный список (ol) применяют, когда важен порядок шагов:

  1. Спланировать структуру документа
  2. Разметить блоки и содержимое
  3. Добавить стили и интерактивность

Список определений

HTML
Язык, определяющий структуру веб-документа.
CSS
Набор правил оформления внешнего вида страницы.
JavaScript
Язык программирования для добавления интерактивности.
